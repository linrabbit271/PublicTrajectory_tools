# ✈️ 物流轨迹集成工具箱 (PublicTrajectory_tools)

**基于 PyQt6 与 高速 Selenium 异步双流驱动的高级空海运/跨境电商物流轨迹全景流转集成工具箱。**

本工具箱专为外贸、跨境电商物流（如 Temu、独立站）以及国际空海运货代行业打造。提供集数据清洗、API 劫持、自动化提单跟踪（Tracking）、海关扣件分流及 OCR 图像智能归档于一体的数字化办公全景解方案。

---

## 🛠️ 核心工作节点与组件矩阵

工具箱采用**多线程异步隔离架构**，彻底杜绝主界面因数据处理卡死，完美覆盖物流全链路闭环流转：

### 1. 全景流转节点
* **1、查时间 (时序校对)**：实时对接各大航司及货站时序节点，多线程校对航班实际离港/到港时间。
* **2、做轨迹文档 (数据清洗)**：支持多仓智能融合，对杂乱的原始日志执行全自动分块切片，秒级生成标准化格式。
* **3、上传文档**：自动化批处理单据上载，无缝衔接主流 TMS 数字化系统与共享数据库。
* **4、更新共享表**：内置二级排序机制，按计划起飞时间与提单号升序规整，实现一键无缝粘贴或 Excel 导出。
* **5、文件看板 (明细看板)**：开启经典斑马线隔行变色及全居中对齐，打造 Excel 级的高级空气呼吸感看板。

### 2. 🧩 扩展组件矩阵 (极速自动化引擎)
工具箱内置了针对各大主流航司、电商平台及二级货站的专属降维打击补件：
* **✈️ 航司/货站高阶黑客提取**：
    * **国泰航空 (Cathay 160)**、**AirZeta (994)**、**中华航空 (CAL 297)**：物理隔离系统焦点争夺，秒级实现输入框自动灌入与多标签页并发查询。
    * **HACTL 极速提取器**：利用内存 JavaScript 异步 Fetch 注入技术，强行接管并劫持货站底层 API JSON 数据流，绕过高频风控。
* **🛒 跨境平台Scraper**：
    * **Temu 运单提取**、**RCL 提取器**、**装货时间提取**等，快速捕捉电商前台轨迹与封签状态。
* **📦 报关风控分流器**：
    * **扣件分流 (包裹号/箱号)**：支持批量混杂文本输入，Pandas 高速 Boolean 索引匹配。在目标路径下自动克隆原始 Excel 格式样式并归档至独立文件夹。
* **📧 数字化图像后装**：
    * **邮件转换器**、**图片批量克隆分发**（微雕复选框轮廓线，支持按揽收、起飞、目的地等 6 大节点自动建档分发）。
    * **拆箱 OCR 智能提取器**：内置 Qt 原生底层拖拽协议（Drag & Drop），直接拖入 `.txt` 文本，毫秒级提取正常单号并归类异常图片。

## 🚀 系统架构与技术风控指标

* **数据脱毒净化**：所有大文本输入区统一采用 `splitlines()` 保底清洗，100% 物理绞杀 Windows 剪贴板夹带的隐藏回车符 `\r`，防止自动化填单与 Pandas 索引发生漏切。
* **异步线程隔离**：界面渲染与底层网络请求、Selenium 电控焦点、文件读写（`openpyxl` / `shutil`）完全独立。
* **免资产本地化**：新版完全基于 PyQt6 原生 Canvas 与样式表绘制，无任何外部图片依赖，体积精简，启动迅捷。
* **安全合规机制**：内置双节点（Gitee / GitHub）安全授权校验（License Filter）以及用户隐私免责弹窗阻断。

## 📦 部署与分发指南

本工具箱已通过 PyInstaller 进行了**工业级单文件（`--onefile`）**编译。

1.  **单文件无损运行**：分发时只需一个极其干净的 `物流轨迹集成工具箱.exe`。
2.  **配置自动理**：程序首次双击启动时，会自动在 EXE 同级目录下智能初始化 `browser_config.json` 路径配置文件。
3.  **原生闪屏**：双击启动后，由系统 C++ 内核完成解压释放，并立即唤起 PyQt6 原生深蓝科技感闪屏（`QSplashScreen`），进度条平滑步进，加载体验极佳。
